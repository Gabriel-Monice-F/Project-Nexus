🚀 Nexus Report
Nexus Report is an Operational Intelligence dashboard designed to transform raw SQL data into actionable insights. It centralizes ticket KPIs, ABC Curve analysis, and features a dynamic Training Matrix (E1-F4) to track module implementation progress across clients in real-time.

📊 Key Features
Operational Dashboard: Real-time visualization of KPIs and technical performance indicators.
Dynamic Training Matrix: A dedicated "Retaguarda" matrix tracking specific modules (E1, E2, E3, F1, F2, F3, F4).
Intelligent Status Coding: Solid, high-contrast color coding (Emerald for Completed, Blue for In-Progress, Orange for Pending, and Slate for Not Started).
ABC Curve Analysis: Automatic classification of client demands and ticket volume.
Professional Export: Built-in functionality to export reports to PDF using html2canvas and jsPDF.
Responsive Tabbed Navigation: Optimized interface with horizontal scroll support for managing multiple modules.

🛠️ Tech Stack
Frontend: Tailwind CSS for a clean, modern "Glassmorphism" UI.
Charts: Chart.js for interactive data representation.
Logic: Vanilla JavaScript (ES6+) for high-performance data filtering and DOM manipulation.
Documentation/Export: html2canvas & jsPDF.

📖 How the Training Matrix Works
The matrix is designed for absolute clarity. It maps Clients vs. Modules. Even if no record exists in the database for a specific module, the system automatically renders a "Not Started" badge in solid gray. This ensures managers can immediately identify implementation gaps.
